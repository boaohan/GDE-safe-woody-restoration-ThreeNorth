
# Outputs Folder

Generated tables and figures can be written here during local runs. Large outputs are intentionally ignored by `.gitignore` and should be archived with the data DOI if they are needed for reproducibility.

Key expected outputs include:

- hydroclimatic LAI carrying-capacity tables
- GDE-constrained safe-LAI tables
- candidate libraries and Pareto fronts
- final best-compromise restoration tables
- QA/QC reports
- sensitivity-analysis reports
- GAM marginal-response figures
- RF/SHAP attribution tables and figures
