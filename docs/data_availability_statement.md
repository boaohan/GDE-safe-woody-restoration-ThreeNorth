
# Data Availability Statement Draft

All code used to reproduce the analyses is available in the GitHub repository associated with this release and archived with a persistent DOI at Zenodo: `10.5281/zenodo.XXXXXXX`.

The processed class-level datasets, derived GDE and GWSA layers, model outputs, QA/QC tables, sensitivity-analysis tables, RF/SHAP attribution outputs, GAM input/output tables, and aggregated field-support tables are archived in a public data repository with DOI: `10.5281/zenodo.YYYYYYY`.

Public remote-sensing and reanalysis datasets were accessed through Google Earth Engine, including TerraClimate, MODIS LAI/NDVI/NPP/land cover products, ERA5-Land, SRTM, and JRC Global Surface Water. Product IDs and processing scripts are provided in this repository.

During peer review, data and code can be shared with reviewers upon request if the journal prefers DOI activation after acceptance.
